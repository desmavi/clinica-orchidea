# Clinica Orchidea

Appointment booking system for Clinica Orchidea, a medical clinic based in Genoa, Italy.

## Demo

- **App**: https://clinica-orchidea.vercel.app/
- **API Docs**: https://clinica-orchidea-api.vercel.app/docs

## Stack

- **Frontend**: React + TypeScript 
- **Backend**: Python + FastAPI
- **Database**: Supabase (PostgreSQL)
- **Email**: Resend

## Project Structure

```
├── backend/          # FastAPI REST API
├── frontend/         # React SPA
└── database/         # SQL schema
```
